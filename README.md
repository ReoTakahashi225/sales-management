# 売上実績管理システム

コスモステクノ静岡 第21期（2025年10月～2026年9月）の売上実績を管理するWebアプリケーションです。

## 機能

- **ダッシュボード**: 年間売上、粗利、月次推移グラフなどの概要表示
- **社員売上管理**: 社員とBPの売上データを一元管理
- **BP原価管理**: ビジネスパートナーの原価データ管理
- **支出管理**: 月次の給与、保険料、個人経費の管理
- **目標管理**: 月次売上目標の設定とグラフ表示
- **月次報告**: 月ごとの売上・支払・粗利レポート
- **データ管理**: JSON形式でのデータインポート/エクスポート

## 技術スタック

- HTML5 / CSS3 / JavaScript (Vanilla)
- Chart.js (グラフ表示)
- LocalStorage (データ永続化)

## デプロイ

### Vercel

このプロジェクトはVercelにデプロイ可能です。

```bash
# Vercel CLIをインストール
npm i -g vercel

# デプロイ
vercel
```

または、GitHubにプッシュしてVercelダッシュボードからインポートすることもできます。

## ローカル実行

静的サイトなので、HTTPサーバーで実行できます。

```bash
# Pythonの場合
python -m http.server 8000

# Node.jsの場合
npx serve
```

その後、ブラウザで `http://localhost:8000` を開きます。

## データ構造

データはLocalStorageに保存され、以下の構造を持ちます：

```json
{
  "employees": [...],
  "partners": [...],
  "expenses": {...},
  "targets": {...},
  "savedAt": "ISO 8601形式の日時"
}
```
